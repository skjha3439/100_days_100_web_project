# My Notes ✏️

A clean and minimal note-taking web app built with HTML, CSS, and JavaScript.  
Write, style, and save your notes effortlessly — with customizable text size, color, and the ability to download notes as a PDF.

## Features
- 📝 Create and edit notes directly in the browser
- 🎨 Change text size and color interactively
- 💾 Save your notes as a styled PDF
- 💻 Auto-save notes in local storage — never lose progress
- 🧊 Smooth, modern UI with light transparency and handwriting font
- 🌈 Responsive and aesthetic design for all screen sizes

## Technologies Used
- **HTML** — for structure  
- **CSS** — for styling and layout (with transparency & glassmorphism)  
- **JavaScript** — for interactivity, local storage, and PDF saving  

## How to Run
1. Download or clone the repository  
2. Open `index.html` in any modern web browser  
3. Start writing your notes and explore the styling options ✨
