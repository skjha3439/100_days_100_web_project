# 🎨 Color Palette Generator

A simple, responsive, and modern web app that generates beautiful random color palettes with their HEX codes.  
Built using HTML, CSS, and JavaScript.

## Features
- Generates 4–5 random colors at once  
- Click any color to copy its HEX code  
- Smooth hover and fade animations  
- Fully responsive design for all screen sizes  
- Minimal, modern, and visually appealing interface  

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## How to Run
1. Download or clone the repository  
2. Open the `index.html` file in your web browser  
3. Click **"Generate Palette"** to create random color palettes  
4. Click on any color box to copy the HEX code  
5. Enjoy experimenting with beautiful colors!
